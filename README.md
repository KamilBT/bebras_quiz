# bebras_quiz
A quiz game for tasks that are in Bebras competition.

This project requires node.js.

Dependencies are:
  {
    "express": "^4.17.1",
    "moment": "^2.24.0",
    "socket.io": "^2.4.0"
  }
  
  
To run project, use npm start. It will create localhost server on port 3000 where you can preview this project.
